# Day 14 - Sentiment Analysis using Hugging Face Transformers

## Objective

The objective of this project is to perform sentiment analysis on customer product reviews using a pre-trained Hugging Face Transformer model.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Hugging Face Transformers
- PyTorch
- Regular Expressions

## Workflow

1. Load the customer reviews dataset.
2. Clean the text using Regular Expressions.
3. Load a pre-trained Hugging Face sentiment analysis model.
4. Predict the sentiment of every review.
5. Store predicted sentiment and confidence score.
6. Visualize sentiment distribution.
7. Export the final dataset as CSV.

## Files

- nlp_sentiment_wrapper.ipynb
- sentiment_analysis_results.csv
- README.md
- requirements.txt

## Output

The final dataset contains:

- Cleaned Text
- Predicted Sentiment
- Confidence Score
