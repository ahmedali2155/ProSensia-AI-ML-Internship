# Day 1 – Data Cleaning with Pandas & NumPy

## Objective

The objective of this project is to clean a corrupted Titanic dataset using Pandas and NumPy by handling missing values, detecting and treating outliers, cleaning inconsistent data, and exporting a machine-readable dataset.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## Tasks Completed

- Loaded the corrupted Titanic CSV dataset
- Handled missing (NaN) values
- Detected and handled outliers using the IQR method
- Cleaned inconsistent text data
- Used Pandas vectorized operations instead of DataFrame row loops
- Exported the cleaned dataset as a new CSV file
- Followed PEP-8 coding standards

## Project Files

- `data_cleaning.py` – Python script for data cleaning
- `cleaned_titanic.csv` – Cleaned dataset
- `requirements.txt` – Required Python libraries

## Output

The cleaned dataset is saved as:

```
cleaned_titanic.csv
```
