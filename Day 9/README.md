# Day 9 – SMOTE and Feature Importance

## Project Objective

The objective of this project is to identify class imbalance, apply SMOTE to the training dataset, retrain a Random Forest classifier, evaluate model performance, and explain predictions using Feature Importance.

---

## Dataset

The project uses the cleaned and One-Hot Encoded Titanic dataset created during Week 1.

---

## Tasks Performed

- Analyzed class distribution
- Applied SMOTE on the training dataset
- Trained a Random Forest classifier
- Evaluated the model using the testing dataset
- Compared pre-SMOTE and post-SMOTE F1-scores
- Extracted Feature Importance
- Created a Top 10 Feature Importance chart
- Exported the trained model

---

## Results

Pre-SMOTE F1 Score: **0.7972**

Post-SMOTE F1 Score: **0.7973**
