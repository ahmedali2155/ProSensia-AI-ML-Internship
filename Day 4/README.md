# Day 4 – Feature Engineering Pipeline

## Objective

Prepare a machine-learning-ready dataset by extracting datetime features and converting categorical variables into numerical features using One-Hot Encoding.

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Completed

- Loaded `final_clean_v2.csv`
- Converted dates using `pd.to_datetime`
- Extracted Month, DayOfWeek, and Is_Weekend
- Applied One-Hot Encoding with `drop_first=True`
- Exported `model_ready_features.csv`
