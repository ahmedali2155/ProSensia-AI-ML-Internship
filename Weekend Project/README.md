# Weekend Project – Exploratory Data Analysis (EDA)

## Project Overview

This project is part of the ProSensia AI/ML Internship Weekend Project.

The objective is to clean a raw e-commerce sales dataset, perform Exploratory Data Analysis (EDA), visualize important trends, and extract meaningful business insights.

---

## Dataset

The dataset contains e-commerce sales information including:

- Customer Information
- Product Categories
- Revenue
- Profit
- Discounts
- Shipping Details
- Payment Methods
- Order Status

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Tasks Performed

- Loaded the Excel dataset
- Explored dataset structure
- Handled missing values
- Removed duplicate records
- Checked and corrected data types
- Performed Exploratory Data Analysis (EDA)
- Created correlation heatmap
- Created distribution plots
- Created box plots
- Created bar charts
- Created scatter plots
- Extracted business insights
- Exported cleaned dataset

---

## Key Business Insights

- Revenue distribution was analyzed.
- Profit distribution was explored.
- Product category performance was compared.
- Revenue and profit relationship was visualized.
- Correlation between numerical variables was examined.
